About the Malaysian Amateur Radio Frequency Database

A community-driven, open-source project providing curated simplex, repeater, and PMR446 frequencies for Malaysian amateur radio operators.

Our Mission

The Malaysian Amateur Radio Frequency Database was created by 9M2PJU to provide a single, reliable, and up-to-date reference for amateur radio frequencies across Malaysia. Before this project, Malaysian operators had to search through scattered PDFs, forums, and outdated lists to find repeater frequencies, CTCSS tones, and channel allocations. Our goal is to make this information freely accessible, searchable, and ready for immediate field use.

What's in the Database

The database contains over 180 curated frequency entries covering:

  • VHF repeaters (144–148 MHz) across all Malaysian states
  • UHF repeaters (430–440 MHz) with DMR and FM modes
  • Simplex calling channels and VHF channel allocations
  • APRS (Automatic Packet Reporting System) on 144.390 MHz
  • PMR446 — all 16 license-free channels (446 MHz)
  • Repeater coordinates with Google Maps links to site locations
  • CTCSS/DCS tone codes for each repeater
  • MARTS and ASTRA linking network information

CHIRP-Compatible

The entire database is available as a CHIRP-compatible CSV file (9M2PJU.csv) that can be directly imported into CHIRP radio programming software. This means you can program your Baofeng, Yaesu, Icom, Kenwood, or Wouxun radio with all Malaysian frequencies in minutes — no manual entry required.

Coverage by State

The database includes repeaters and frequencies from every Malaysian state:

  • Kuala Lumpur & Selangor — Bukit Lanjan repeaters (9M2RKL, 9M2RKK)
  • Pahang — Gunung Ulu Kali, Bukit Brinchang, Cameron Highlands
  • Penang — Bukit Bendera (Penang Hill) repeaters
  • Melaka — Bukit Beruang, Gunung Ledang
  • Johor — Gunung Pulai, Johor Bahru, Batu Pahat
  • Kedah — Gunung Jerai, Gunung Keriang
  • Perak — Bukit Larut, Lenggong, Taiping
  • Perlis — Kangar
  • Terengganu — Marang, Besut, Kijal
  • Kelantan — Kota Bharu, Tumpat, Gua Musang
  • Sabah — Kota Kinabalu, Gunung Kinabalu, Sandakan, Tuaran
  • Sarawak — Kuching, Miri, Borneo Highlands, Serian

Open Source & Community-Driven

This project is open-source under the GNU General Public License 3.0 (GPL-3.0). The source code and frequency data are hosted on GitHub. We welcome contributions — if you find a new repeater, a frequency change, or an error, please submit a pull request or open an issue.

About 9M2PJU

9M2PJU is a Malaysian amateur radio operator and the creator of this frequency database. The project began as a personal reference and grew into a community resource used by operators across Malaysia. 9M2PJU also maintains hamradio.my, a Malaysian amateur radio community website.

Acknowledgements

Special thanks to 9M2EDU, MARTS (Malaysian Amateur Radio Transmitters' Society), and all the amateur radio operators who contribute frequency data, verify repeater information, and help keep this database accurate and up to date.

Technical Details

This website is a Progressive Web App (PWA) built with vanilla JavaScript, Tailwind CSS, and PapaParse for CSV parsing. It's hosted on GitHub Pages at frequency.hamradio.my. The site works offline once installed, making it ideal for field use where internet connectivity may be limited. All frequency data is pre-rendered into the HTML for search engine optimization and accessibility.
