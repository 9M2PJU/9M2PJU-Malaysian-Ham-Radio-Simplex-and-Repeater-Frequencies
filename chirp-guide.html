CHIRP Programming Guide for Malaysian Ham Radio

Learn how to program all Malaysian amateur radio frequencies into your walkie-talkie or mobile radio using CHIRP software and the 9M2PJU CSV file. Works with Baofeng, Yaesu, Icom, Kenwood, and Wouxun radios.

What is CHIRP?

CHIRP is a free, open-source radio programming tool that supports over 100 different amateur radio models from manufacturers like Baofeng, Yaesu, Icom, Kenwood, and Wouxun. Instead of manually entering frequencies one by one using your radio's keypad, CHIRP lets you program all channels at once via a spreadsheet-like interface. It's the standard tool used by amateur radio operators worldwide.

The 9M2PJU CSV file is pre-formatted to be directly compatible with CHIRP, containing all the fields CHIRP expects: frequency, offset, duplex direction, CTCSS/DCS tones, mode, power, and location comments.

Prerequisites

  • A computer running Windows, macOS, or Linux
  • A USB programming cable compatible with your radio (e.g., Kenwood KPG-22 cable for Baofeng)
  • Your amateur radio or walkie-talkie (Baofeng UV-5R, Yaesu, Icom, etc.)
  • A valid AROC license if you plan to transmit on amateur radio bands

1Download and Install CHIRP

  1. Go to chirp.danplanet.com
  2. Click Download and choose the version for your operating system
  3. For Windows: download the installer and run it
  4. For macOS: download the .dmg file and drag CHIRP to your Applications folder
  5. For Linux: install via your distribution's package manager or download the tarball

Note for Windows users: You may need to install the Prolific or FTDI USB-to-serial driver for your programming cable. Many Baofeng cables use counterfeit Prolific chips — you may need to use an older driver version.

2Download the 9M2PJU CSV File

  1. Go to frequency.hamradio.my
  2. Click the CSV download button in the toolbar
  3. Or download directly from https://frequency.hamradio.my/9M2PJU.csv
  4. Save the file to a location you can easily find (e.g., your Desktop)

The CSV file contains all 189 frequency entries including repeaters, simplex channels, APRS, and PMR446 channels, all pre-formatted for CHIRP.

3Connect Your Radio

  1. Power OFF your radio
  2. Connect the programming cable to your radio's Kenwood-style jack (SPK/MIC ports)
  3. Connect the USB end to your computer
  4. Power ON your radio
  5. Ensure the radio volume is turned up (some radios won't program if volume is at zero)

4Read From Your Radio

  1. Open CHIRP
  2. Go to Radio > Download From Radio
  3. Select the correct Port (COM3 on Windows, /dev/ttyUSB0 on Linux, etc.)
  4. Select your radio manufacturer and model
  5. Click OK and wait for the download to complete

This reads the current channel data from your radio so CHIRP knows the layout. If this is a new radio, it will show default/empty channels.

5Import the CSV File

  1. Go to File > Import
  2. Browse to and select the 9M2PJU.csv file you downloaded
  3. CHIRP will show a preview of the channels to import
  4. Select All or choose specific channels you want to import
  5. Click OK to import
  6. The frequencies will appear in the CHIRP spreadsheet view

You can review and edit any channel before uploading. For example, you might want to adjust the power level or skip certain channels if your radio has limited memory slots.

6Upload to Your Radio

  1. Go to Radio > Upload To Radio
  2. Confirm the port and radio model
  3. Click OK to begin the upload
  4. Do not disconnect the cable or power off the radio during upload
  5. Wait for the progress bar to complete
  6. Power off the radio, disconnect the cable, and power back on

Your radio is now programmed with all Malaysian amateur radio frequencies! Test by tuning to the simplex calling frequency 145.000 MHz or your nearest repeater.

Troubleshooting Common Issues

"Radio did not respond" error

  • Make sure the radio is powered ON and volume is up
  • Try a different USB port (use USB 2.0 if available)
  • Check that you selected the correct COM port
  • For Baofeng cables with Prolific chips, install the Prolific 3.2.0.0 driver (older version that works with counterfeit chips)
  • Try removing and reinserting the cable while CHIRP is running

Wrong radio model selected

If you have a Baofeng UV-5R variant (UV-5RA, UV-5RE, UV-5R V2+, etc.), try selecting Baofeng UV-5R as the model. Most variants are compatible with the base UV-5R selection.

Cable not detected

  • On Windows, check Device Manager > Ports (COM & LPT) for the cable
  • On Linux, run dmesg | grep tty to find the device
  • On macOS, check /dev/cu.* or /dev/tty.*
  • Install FTDI or Prolific drivers if the cable doesn't appear

Supported Radios

The 9M2PJU CSV file works with any radio supported by CHIRP. Popular models used by Malaysian amateur radio operators include:

  • Baofeng: UV-5R, UV-5RA, UV-5RE, UV-82, BF-F8HP, UV-K5
  • Yaesu: FT-60R, FT-65R, FT-25R, FT-70DR, VX-6R, VX-7R
  • Icom: IC-V80, IC-T70, IC-W32A, ID-51
  • Kenwood: TH-F6A, TH-G71, TK-270G
  • Wouxun: KG-UV8D, KG-UV9D, KG-UV3D
  • AnyTone: AT-D878, AT-D868

For the complete list, see the CHIRP Supported Radios page.
