Complete Malaysian Amateur Radio Frequency List

The Malaysian Amateur Radio Frequency Database by 9M2PJU is a curated, searchable collection of 206 frequencies for amateur radio operators in Malaysia. It covers VHF (144–148 MHz), UHF (430–440 MHz), and PMR446 (446 MHz) bands, including repeater channels with CTCSS/DCS tones, simplex calling channels, APRS (144.390 MHz), and all 16 PMR446 license-free channels. The database is CHIRP-compatible and can be downloaded as a CSV file for direct programming of Baofeng, Yaesu, Icom, Kenwood, and Wouxun radios.

Malaysian repeaters listed include stations in Kuala Lumpur, Selangor, Pahang, Penang, Melaka, Johor, Kedah, Perak, Perlis, Terengganu, Kelantan, Sabah, and Sarawak. Major repeater sites include Bukit Lanjan, Gunung Ulu Kali, Bukit Brinchang, Gunung Ledang, Bukit Bendera (Penang Hill), Gunung Jerai, Gunung Kinabalu, and Borneo Highlands. Repeaters are linked via MARTS (Malaysian Amateur Radio Transmitters' Society) and ASTRA Transnasional linking networks.

How to Use This Database

Use the search box to find frequencies by callsign, location, or tone. Filter by band (VHF, UHF) or type (Simplex, Repeaters). Click "Near Me" to sort repeaters by distance from your location. Download the CSV file and import it into CHIRP radio programming software to program your walkie-talkie or mobile radio. See our CHIRP programming guide for step-by-step instructions.

For license-free personal use, see our PMR446 guide. For information on amateur radio bands in Malaysia, see our Malaysian ham radio bands guide. For frequently asked questions, visit our FAQ page. To learn more about this project, visit the about page.
